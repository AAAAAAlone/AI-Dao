# 玄空数术 · 排盘视觉工具

一个专注于传统术数排盘与界面呈现的前端项目。它把八字、六爻和择日日历中的高密度信息，组织成清晰、克制、适合桌面与移动设备的视觉系统。

## 包含内容

### 八字排盘

- 公历、农历与闰月输入
- 出生地、时区、经度与真太阳时校正
- 四柱、十神、藏干、纳音、十二长生与旬空
- 大运、流年、流月和神煞的时间轴展示
- 盘面摘要复制与浏览器本地保存

### 六爻排盘

- 数字、时间、卦名与手动定爻
- 本卦、变卦、六亲、六神、伏神、世应与动爻
- 朱砂红、纸张肌理与高信息密度的双栏盘面
- 图片导出、收藏和浏览器本地历史记录

### 择日日历

- 月历与每日干支信息
- 十二长生、地支关系与五行信息
- 意向筛选和关系视图
- 响应式桌面与移动端布局

## 设计语言

界面以米白纸色、墨色与朱砂红为主，结合印章、细网格、纵向题签和中英文字标。信息层级优先服务于快速核对：核心盘面醒目，计算口径与辅助信息保持可追溯，交互控制收敛在边缘区域。

## 本地运行

需要 Node.js 20 或更高版本。

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:3000`。

生产构建：

```bash
npm run build
npm run preview
```
