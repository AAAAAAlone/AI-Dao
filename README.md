   起卦流程与规则严格符合传统，结果可靠。

3. **支持现在更常用的报数字等方式**  
   现代更常见的报数起卦、时间起卦等常用方式均支持。

4. **基础的收藏与登录**  
   Web 与手机都能用；支持免注册、仅本地缓存即可使用。

5. **可切换万年历 + 用神刑冲合克择日**

   择日万年历专门为择日场景设计，并附带新手向择日基础说明，方便入门。
